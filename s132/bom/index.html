            # Mounting hardware

            Part Quantity Description
            1

            M4x10mm Hex Socket Screw

            The M4 socket screw is used to fix the S-132 to the clear acrylic back of the IMSAI80080/Cromemco Z-1. You will need to tap or drill a hole to secure the M4 socket screw.
            1

            M4 Hex Nut Nylon

            The nut can be used as a spacer between the S-132 PCB and the acrylic back of the IMSAI80080/Cromemco Z-1
            1

            M3x5mm Zinc Machine Screw & Washer

            The M3 machine screw, washer and nut are used to secure the LM7805 to the S-132 PCB, to aide thermal conduction with the PCB.
            1
